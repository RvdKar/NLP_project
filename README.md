# An NLP analysis of the subreddit r/AmItheAsshole

## Data Setup:
Place raw data files (.ndjson) in the `/data` directory.

## Installation
Ensure you have uv installed (see https://docs.astral.sh/uv/#installation) and run `uv sync`.
